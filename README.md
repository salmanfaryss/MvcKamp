📚 MVC Öğrenme Yolculuğum

Bu proje, değerli hocam Murat Yücedağ'ın YouTube üzerinde yayınladığı yaklaşık 100 videoluk ASP.NET Core MVC eğitim serisini takip ederek geliştirdiğim bir çalışmadır.

Eğitim boyunca sadece temel MVC yapısını değil, aynı zamanda profesyonel projelerde kullanılan birçok önemli konuyu öğrenme fırsatı buldum:

✅ N Katmanlı Mimari (Layered Architecture)
✅ CRUD İşlemleri
✅ ASP.NET Core MVC
✅ Entity Framework Core
✅ Authentication (Kimlik Doğrulama)
✅ Authorization (Yetkilendirme)
✅ Hata Yönetimi ve 404 Sayfaları
✅ View Components
✅ Layout Yönetimi
✅ Routing Yapıları
✅ Repository Pattern
✅ Veritabanı İşlemleri ve İlişkiler

Bu eğitim sayesinde hem teorik bilgilerimi hem de pratik geliştirme becerilerimi önemli ölçüde geliştirdim. Her bölüm açık, anlaşılır ve uygulamalı olarak anlatıldığı için MVC dünyasına giriş yapmak isteyenler için oldukça faydalı bir kaynak olduğunu düşünüyorum.

MVC öğrenmeye yeni başlayan, ASP.NET Core teknolojilerine adım atmak isteyen veya gerçek bir proje üzerinden ilerlemek isteyen tüm arkadaşlarıma bu eğitim serisini gönül rahatlığıyla tavsiye ederim.

Bu süreçte bana yeni bakış açıları kazandıran ve bilgi birikimini paylaşan hocam Murat Yücedağ'a teşekkür ederim.

🚀 Öğrenmeye, üretmeye ve kendimi geliştirmeye devam ediyorum.
